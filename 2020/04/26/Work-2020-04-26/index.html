Work 2020.04.26

准备阶段

  1. node -v //查看node的版本
  2. git version //查看git的版本
  3. npm install -g hexo-cli //安装 hexo
  4. hexo init Myblog //hexo init 生成需要的文件，电脑上就能看到Myblog这个文件夹诞生了
  5. cd Myblog //进入Myblog文件夹
  6. hexo g //生成网站页面， Myblog文件夹中就多了public文件夹
  7. hexo s //生成网址，在浏览器中查看这个网站

部署

  1. npm install hexo-deployer-git –save //安装 hexo-deployer-git
  2. npm list hexo-deployer-git //检查是否安装成功
